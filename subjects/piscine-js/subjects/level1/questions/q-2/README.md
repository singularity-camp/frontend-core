# Two Sum

Напишите программу, которая считает сумму двух чисел.

#### ## Sample input 1:

```bash
1, 2
```

#### ## Sample output 1:

```bash
3
```

#### ## Sample input 2:

```bash
-1, 3
```

#### ## Sample output 2:

```bash
2
```

#### ## Sample input 3:

```bash
0, 7
```

#### ## Sample output 3:

```bash
7
```

#### Sample input 4:

```bash
-5, 7
```

#### Sample output 4:

```bash
2
```

Ответ:

```jsx
function sum(a, b) {
  return a + b;
}
```
