# More or less

Coздайте 4 функции:

- `more` принимает 1 аргумент и прибавляет 1 к нему
- `less` принимает 1 аргумент и вычитает 1 от него
- `add` принимает 2 аргумента и прибавляет их
- `sub` принимает 2 аргумента и вычитает их

## More

#### ## Sample input 1:

```bash
1
```

#### ## Sample output 1:

```bash
2
```

## Less

#### ## Sample input 1:

```bash
1
```

#### ## Sample output 1:

```bash
0
```

## Add

#### ## Sample input 1:

```bash
1, 2
```

#### ## Sample output 1:

```bash
3
```

## Sub

#### ## Sample input 1:

```bash
1, 2
```

#### ## Sample output 1:

```bash
-1
```

Ответ:

```jsx
function more(a) {
  return a + 1;
}

function less(a) {
  return a - 1;
}

function add(a, b) {
  return a + b;
}

function sub(a, b) {
  return a - b;
}
```
