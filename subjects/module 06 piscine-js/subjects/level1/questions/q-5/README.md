# Id

Создайте следующие функции:

- `id`, который принимает один аргумент и возвращает его
- `getLength`, который принимает строку и возвращает ее длину

## id

#### ## Sample input 1:

```bash
1
```

#### ## Sample output 1:

```bash
1
```

## getLength

#### ## Sample input 1:

```bash
oPT6ZmQcPEpNXAgx
```

#### ## Sample output 1:

```bash
16
```

Ответ:

```jsx
function id(a) {
  return a;
}

function getLength(a) {
  return a.length;
}
```
